# Redstone Town Client · 红石镇客户端

整合包仓库，包含以下目录：

| 目录 | 内容 |
|------|------|
| `mods/` | 模组文件 |
| `config/` | 配置文件 |
| `resourcepacks/` | 资源包 |
| `shaderpacks/` | 光影包 |
| `PCL/` | PCL 启动器相关文件 |

## 获取更新

使用 [红石镇客户端更新器](https://github.com/Mashiro-Neri/RSTC-Updater) 自动检测并更新。

1. 从 [RSTC Updater Release](https://github.com/Mashiro-Neri/RSTC-Updater/releases) 下载 `update_modpack.ps1` 和 `update_modpack.bat`
2. 放入版本文件夹内（一般在 `.minecraft\versions\红石镇客户端\`），放在里面可以自动获取路径
3. 双击 `update_modpack.bat`，选择「检查整合包更新」
4. 程序会自动比对本机版本与最新 Release，完成下载与文件同步
